# aws

A collection of AWS project demos and learning resources, organized by service or topic.

## Folders

| Folder | Description |
|--------|-------------|
| [cicd/](cicd/) | End-to-end CI/CD pipeline demo using CodeCommit, CodeBuild, CodeDeploy, and EC2. Includes the app, pipeline configs, deployment scripts, architecture diagrams, and presentation slides. |
| [containers/](containers/) | Containers on AWS demo — Docker + ECR + ECS Fargate. Includes a containerized HTML app (Dockerfile), 32-slide presentation, full demo script, architecture diagrams, instructor guide, attendee guide, and a comprehensive 16-section beginner guide covering deployment, scaling, and production best practices. |
